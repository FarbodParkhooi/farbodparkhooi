## 👨‍💻 About Me

<p align="center">
  <img src="https://media.giphy.com/media/qgQUggAC3Pfv687qPC/giphy.gif" width="250" alt="Coding gif" align="right" />

I'm a **professional programmer** based in Iran with deep expertise across multiple domains:

- **Core Languages**: Python, C, C++ — but broadly proficient across most programming languages
- **Specialties**: AI/ML programming, Computer Vision, App Development, Microcontroller Programming
- **Hardware Design**: Professional-level SolidWorks (3D/mechanical) and Altium (PCB/electronics)
- **Philosophy**: Build everything from scratch. No pre-built solutions. Deep understanding over quick fixes.

🔭 **Currently working on**:
- **Agaah Counseling Center** — Django-based appointment booking system (multi-role, Persian UI, video session integration)
- **DIY AR Goggles** — Video See-Through headset prototype with custom PCB design and computer vision pipeline
- **CIFAR-10 GAN, and cGAN** — GAN and cGAN models on PyTorch
- **F35-view** — A simulated F35 HUD including CV features

🌱 **Learning**: Advanced GAN architectures, Voice Generation (TTS, vocoders), Computer Vision

💬 **Ask me about**: AI/ML, Embedded Systems, PCB Design, 3D Printing, Building from scratch

⚡ **Fun fact**: I designed a custom STM32F103C8T6 flight controller entirely from scratch — no Betaflight, no Cleanflight, just my own code.
